# Quickly flex Instalation

## Usage
* Update configs in `./quickly_install/flex/install_from_release.sh`
* Run `bash ./quickly_install/flex/install_from_release.sh`
* Run sth like:
```shell
ln -s ${HOME}/bin/flex-2.6.4/bin/flex ${HOME}/bin/flex
ln -s ${HOME}/bin/flex-2.6.4/bin/flex++ ${HOME}/bin/flex++
```
